Surgimento da Intenert

(Voltar)

A internet surgiu nos Estados Unidos na década de 1960, durante a Guerra Fria, com a criação da ARPANET , uma rede de troca de informações desenvolvida para fins militares.

O primeiro envio de um pacote pela internet aconteceu em 29 de outubro de 1969, na Universidade da Califórnia , Los Angeles (UCLA), e foi feito para o Stanford Research Institute. A ARPANET conectava inicialmente quatro universidades, utilizando a comutação de pacotes para transmissão de dados.

A internet chegou ao Brasil em 1988, por meio de uma conexão entre a Universidade Federal do Rio de Janeiro (UFRJ) e a rede da Universidade de Maryland, nos Estados Unidos. A popularização da internet no Brasil ocorreu nos anos 1990, quando ficou mais barata e o número de usuários cresceu exponencialmente.

A internet é atualmente o maior meio de comunicação utilizado no mundo, com mais de 5 bilhões de usuários.

Como funciona o HTML

(Voltar)

O HTML, ou Linguagem de Marcação de Hipertexto, é uma linguagem de marcação que define a estrutura e a formatação de páginas web. O seu funcionamento é o seguinte:

  • O HTML é um arquivo de texto puro com a extensão .html, que é armazenado num servidor.
  • Quando um utilizador digita um endereço de um site, o navegador pede uma cópia do documento HTML da página correspondente ao servidor.
  • O navegador interpreta o código HTML de cima para baixo, de acordo com as instruções das tags e atributos.
  • O navegador estrutura o conteúdo, aplica formatações e estilos, e exibe a página.

O HTML é composto por elementos, como tags e atributos, que delimitam e agrupam partes do conteúdo. As tags indicam ao navegador como apresentar a informação marcada. Por exemplo, é possível usar tags para transformar uma palavra em hiperlink, colocar palavras em itálico, aumentar ou diminuir a fonte, entre outras coisas.

Para garantir que o navegador interprete corretamente o código HTML, é necessário declarar a versão do HTML a ser usada. Para isso, deve-se incluir a instrução no início do documento.

O HTML é usado em diversos contextos, como no desenvolvimento de sites, de aplicativos para dispositivos móveis e de jogos.

O que são Tags?

(Voltar)

Tags HTML são como as palavras-chave que orientam o navegador sobre como exibir o conteúdo de uma página da web. Elas definem a estrutura, a formatação e o significado de cada elemento presente na página.

Imagine que você está escrevendo um livro. As tags HTML seriam como os sinais de pontuação, os títulos dos capítulos, as indicações de parágrafos e as marcações para destacar palavras importantes. Elas fornecem as instruções necessárias para que o texto seja formatado e apresentado de forma correta.

Em outras palavras, as tags HTML:
  • Estruturam o conteúdo: Definindo onde começa e termina um título, um parágrafo, uma lista, uma imagem, etc.
  • Formatam o texto: Aplicando negrito, itálico, sublinhado, alterando o tamanho da fonte, etc.
  • Adicionam elementos multimídia: Inserindo imagens, vídeos, áudio, etc.
  • Cria links: Permitindo a navegação entre diferentes páginas.

Exemplo:

		
	<!DOCTYPE html>
	<html>
	<head>
	    <title>Minha primeira página</title>
	</head>
	<body>
	    <h1>Bem-vindo ao meu site!</h1>
	    <p>Este é um exemplo de parágrafo.</p>
	    <img src="minha_imagem.jpg" alt="Uma imagem bonita">
	</body>
	</html>
